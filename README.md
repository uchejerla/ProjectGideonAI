# Gideon AI Monorepo

Minimal monorepo layout with a desktop app built on Tauri + React + TypeScript.

## Structure

- `apps/desktop`: Tauri shell with a React + TS UI.

## Local development

```bash
cd apps/desktop
npm install
npm run dev
```

To launch the desktop shell:

```bash
npm run tauri:dev
```

## Build

```bash
cd apps/desktop
npm run build
```
